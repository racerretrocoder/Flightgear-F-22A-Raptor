# Lockheed Martin F-22A Raptor

![image](/Screenshots/pic.png)
This is the F-22 add-on for FlightGear!
If you need help: You can join our Discord server temporarily if you want: https://discord.gg/B9WWFcwY2V

## Installation:

To download this aircraft, press [here](https://github.com/racerretrocoder/Flightgear-F-22A-Raptor/archive/refs/heads/master.zip) or the green button and then press 'Download ZIP'.

Once you downloaded the file, move it to a folder where you want to save all of your custom aircraft, if you haven't got one already.
Then, launch FlightGear, go to the Add-ons tab and add the aircraft folder to the `Additional aircraft folders`.

### Example:

Say you stored your airplane in `home/Documents/(CustomAircraftFolder)/F-22`.
Now add the `CustomAircraftFolder` to `Additional aircraft folders` in FlightGear.

Bam! You just achieved air superiority.

If you still have problems installing, you can join our Discord server.

## How to fly

#### Important Keys

- `l` deploys the currently selected counter messures program (see the CMS menu in the SMS page on a MFD to cycle between programs)
- `m` switches the current air-to-air missile
- `n` switches the current air-to-gnd bomb/missile

- `Ctrl+f` Toggles the side baydoors
- `Ctrl+d` Toggles the center baydoors
- `Ctrl+o` Unlock from the currently locked radar target
- `w` switches the triggers mode. guns or missiles
- `n/m` to choose what weapon you shoot when in missile mode
- `e` shoots the selected weapon

There is no key for pickle.

### Radar

The FCR on the MFDs is the radar screen

- `Shift+r` to enable / disable the radar
- `r` changes the radars range.
- `y` locks a target. if you see a target on the FCR, press y to select it, and lock it up

#### Radar Modes:

<table>
    <tr>
        <td>TWS</td>
        <td>Track while scan</td>
        <td>The default radar mode for pretty much any situation</td>
    </tr>
    <tr>
        <td>RWS</td>
        <td>Range while scan</td>
        <td>This mode allows the radar to search 320NM (592km) out!</td>
    </tr>
    <tr>
        <td>ACM</td>
        <td>Dogfight Mode</td>
        <td>Automatically seek and lock targets if positioned in the HUD</td>
    </tr>
    <tr>
        <td>SLR</td>
        <td>Side-Looking Radar</td>
        <td>Raptor has advanced SLR'S at the side of the nose to allow for a crazy scan azimuth of 280 degrees. It's possible to lock and fire Aim-120's at bandits that are roughly on your tail!</td>
    </tr>
</table>
